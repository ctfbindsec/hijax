# Payloads Directory

This directory stores generated HTML/JavaScript implant payloads.

## Files Generated Here:

- `implant_drop_<timestamp>.html` - Generated browser implants for remote deployment

## Purpose:

Implants generated by Hijax are designed for:
- Remote token/cookie exfiltration
- Browser-based session capture
- Integration with ScamTrack or other delivery mechanisms

## Usage:

Generate payloads using:
```bash
python3 hijax_cli.py
# Select option [4] Generate Remote Implant
```

Or directly:
```bash
python3 hijax_remote_implant.py
```

## Deployment:

Serve payloads via:
- Python HTTP server: `python3 -m http.server 8080`
- ScamTrack integration
- Ngrok tunnels for remote access
- Other web hosting methods

## Security Note:

⚠️ **Use only for authorized testing!**

- Deploy only on systems you own or have permission to test
- This directory is excluded via `.gitignore`
- Never use against unauthorized targets
